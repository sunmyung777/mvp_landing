# Re:joice Landing Page

React와 Tailwind CSS로 구축된 Re:joice 브랜드의 랜딩 페이지입니다.

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 이미지 가이드라인

### 로고 이미지
- 파일 위치: `public/images/logo.png`
- 권장 크기: 200x50px (가로x세로)
- 파일 형식: PNG (투명 배경)

로고 이미지를 사용하려면:
1. `logo.png` 파일을 `public/images/` 디렉토리에 위치시킵니다.
2. 이미지는 자동으로 Header 컴포넌트에 로드됩니다.

## 기술 스택

- React
- Vite
- Tailwind CSS
- PostCSS
